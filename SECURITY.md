# Security

## Supported versions

Security fixes are applied to the latest release on the `main` branch. Use tagged releases for production or reproducible research when possible.

## Reporting a vulnerability

Please **do not** open a public issue for security-sensitive reports.

Email the maintainer with:

- A short description of the issue and its impact
- Steps to reproduce (if any)
- Whether you believe it affects PHI, credentials, or supply chain

We aim to acknowledge receipt within a few business days. This project is maintained by a small team; complex issues may take longer to resolve.

## Scope

This repository contains **educational and clinical decision-support** software. It is not a substitute for institutional security review for regulated deployments.
